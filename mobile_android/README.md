# App Adı: BVBarber

## Berber Randevu Uygulaması – Android (Flutter)

### Amaç

Bu proje, bir berber dükkanı için **randevu ve yönetim uygulaması** olarak Flutter kullanılarak geliştirilmektedir. Android platformu için geliştirme bu repo üzerinden ilerleyecek.

---

## 1) 🚀 Proje Hedefleri

- Kullanıcılar uygulamaya kayıt olabilir ve giriş yapabilir.
- Berberler ve müşteriler için CRUD işlemleri:
  - Hizmet ekleme, düzenleme, silme
  - Randevu ekleme, düzenleme, silme
- Rezervasyon sistemi (takvim/slot bazlı)
- Firebase backend ile veri senkronizasyonu
- Push bildirimleri ile randevu hatırlatma
- Play Store üzerinden yayın

---

## 2) 🛠️ Teknolojiler

- Flutter (Dart)
- Firebase:
  - Authentication
  - Firestore Database
  - Storage
  - Cloud Messaging (push notification)
- GitHub (versiyon kontrolü)
- Android Studio / Emulator

---

## 3) 🎨 UI / UX Tasarım Rehberi

Bu uygulamanın tüm tasarım yapısı **Stitch** platformu kullanılarak hazırlanmıştır. Uygulamanın geliştirme sürecinde Flutter arayüzleri, Stitch üzerinde oluşturulan tasarımlara mümkün olduğunca sadık kalacak şekilde geliştirilecektir.

Proje ilerledikçe güncel tasarım görselleri ve ekran referansları bu dokümana eklenecektir.
